Media Gallery Web Stuck Loading

Back to Home

BACKGROUND

This issue presents itself in Media Gallery where it just states "Loading..." endlessly. In the devtool logs, there are no indications of media-gallery being called. This can be caused by the Orion Embedded URL setup being missing or misconfigured.

STEPS

You can reference the full instructions here, in case anything else may be amiss. Contact the #ww-ohai-orion-wfs Slack channel if you need further assistance.


1. Check if the embedded Orion URL is defined:

select * from dm_info
where info_domain = "ORION"
and info_name = "EMBEDDED_BASE_URL"

2. If nothing returns, it needs to be defined. Provide the appropriate URL from the list below determined by region.

insert into dm_info set
info_char = "xxxxx",
info_domain = "ORION",
info_name = "EMBEDDED_BASE_URL",
updt_dt_tm = cnvtdatetime(curdate,curtime3)

3. If row returned, review the list from below and ensure the appropriate URL is defined determined by region.

update into dm_info
set info_char = "xxxxx",
updt_dt_tm = cnvtdatetime(curdate,curtime3)
where info_domain = "ORION" and INFO_NAME = "EMBEDDED_BASE_URL"
