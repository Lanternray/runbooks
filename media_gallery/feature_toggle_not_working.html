Feature Toggles Not Taking Effect

Back to Home

BACKGROUND

Feature Toggles are managed/configured in Cloud. Sometimes a case will arise where Feature Toggles are enabled but not taking effect in Millennium as expected. For example, a client has the "Media Gallery Web" Feature Toggle enabled, but the Legacy version is still loading.

ITEMS FOR INVESTIGATION

  • Workflow to identify what MPage is involved and to reproduce issue
  • Current settings in Cloud. If unable access client's Features, may need to engage Cloud Enablement support to provide what's enabled for client
  • Logs from MPages if applicable

STEPS

From the backend app tier, call apptest to test request 2030001 for an impacted user to see what Feature Toggles are returned

apptest -srv

Request Name/Number: 2030001

REQUEST 1 SRV_STRING (C0) username CERNER

In the prompt, you should only have to provide the request number and username. Leave the other values blank.

Check to see what Feature Toggles are being returned.

If you do not see the one you're needing and confirmed the Feature Toggles enabled, then cycle SCP 2030 (Feature Toggle Management). This should resolve the issue.
