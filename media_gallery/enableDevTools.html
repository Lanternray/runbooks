Enable Inspect Element (DevTools)

Back to Home

BACKGROUND

In many cases, Citrix teams are now implementing Citrix policies which lock down the ability to access DevTools from the MPages which are necessary for troubleshooting.

The following can be performed to allow access to the "Inspect" element in the context menu. This is assuming the account performing these steps has 'modify' privileges for the registry.

STEPS

First, launch CMD prompt as Administrator.

Then run the following:

reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Edge" /v DeveloperToolsAvailability /t REG_DWORD /d 1 /f

This should now allow the Inspect element to be available and/or access to DevTools.

If not, or if permissions to do so are denied, you will need to work with the OHH Citrix group for further assistance to provide access.
